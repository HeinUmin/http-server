## Usage

```
Usage: ./http-server [options]
Options:
  -h, --help               Display this help message
  -p, --http-port=PORT     Set HTTP port, 0 for close
  -s, --https-port=PORT    Set HTTPS port, 0 for close
  -l, --log-level={trace|debug|info|warn|error|fatal}
                           Set log level
```
